## ADDED Requirements

### Requirement: Web-based demo mode
The system SHALL provide interactive web-based demo for potential users.

#### Scenario: User accesses demo
- **WHEN** user visits demo URL
- **THEN** system presents interactive demonstration
- **AND** allows limited transcription trials

#### Scenario: Tutorial mode
- **WHEN** new user starts tutorial
- **THEN** system guides through key features
- **AND** provides sample exercises

### Requirement: Sample use cases
The system SHALL include pre-configured sample scenarios.

#### Scenario: Demo scenarios available
- **WHEN** user browses demo scenarios
- **THEN** system shows common use cases
- **AND** allows trying each with example content
