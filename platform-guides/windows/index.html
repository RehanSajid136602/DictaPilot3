    Windows Setup Guide

    Complete guide for setting up DictaPilot3 on Windows.

    System Requirements

    • Windows 10 or Windows 11
    • Python 3.10 or higher
    • Microphone access
    • Administrator privileges (for some features)

    Quick Setup

    1. Install Python

    Download Python from python.org: - Check "Add Python to PATH" during installation - Install for all users (recommended)

    Verify installation: 

    python --version

    2. Clone Repository

    Using Git: 

    git clone https://github.com/RehanSajid136602/DictaPilot.git
cd DictaPilot

    Or download ZIP from GitHub and extract.

    3. Run Setup Script

    setup\setup_windows.bat

    This installs all required dependencies.

    4. Configure API Key

    Create .env file in project root: 

    echo GROQ_API_KEY=your_api_key_here > .env

    Or use Notepad: 

    notepad .env
    Add: GROQ_API_KEY=your_api_key_here

    5. Grant Microphone Permission

    Windows 10/11: 1. Settings → Privacy → Microphone 2. Enable "Allow apps to access your microphone" 3. Enable for Python or Terminal

    6. Run DictaPilot

    python app.py

    Backend Configuration

    Recommended Settings

    # In .env
HOTKEY_BACKEND=keyboard
PASTE_BACKEND=keyboard

    Hotkey Backends

    keyboard (Recommended): 

    HOTKEY_BACKEND=keyboard
    - Native Windows support - Most reliable - May require admin for some apps

    pynput (Alternative): 

    HOTKEY_BACKEND=pynput
    - Cross-platform - Good fallback - Works without admin

    Paste Backends

    keyboard (Recommended): 

    PASTE_BACKEND=keyboard
    - Direct keyboard simulation - Fast and reliable - Works with most apps

    pynput (Alternative): 

    PASTE_BACKEND=pynput
    - Pure Python implementation - Good compatibility - Slightly slower

    Permissions

    Microphone Access

    Windows 10/11:

    1. Settings → Privacy → Microphone
    2. Enable "Allow apps to access your microphone"
    3. Scroll down and enable for:
    4. Python
    5. Windows Terminal
    6. PowerShell
    7. Command Prompt

    Verify microphone: 

    python -c "import sounddevice; print(sounddevice.query_devices())"

    Administrator Privileges

    Some features may require admin rights:

    When needed: - Global hotkeys in some applications - Keyboard hooks for certain apps - System-wide clipboard access

    How to run as admin: 1. Right-click Command Prompt or PowerShell 2. Select "Run as administrator" 3. Navigate to DictaPilot directory 4. Run python app.py

    Note: Try running as normal user first. Only use admin if needed.

    Troubleshooting

    Hotkey Not Working

    Problem: Pressing F9 does nothing

    Solutions:

    1. Check for conflicts:
    2. F9 may be used by other applications

    3. Try different hotkey: 

      HOTKEY=f10

    4. Try different backend: 

      HOTKEY_BACKEND=pynput

    5. Run as administrator:

    6. Right-click Command Prompt → Run as administrator
    7. Navigate to DictaPilot folder

    8. Run python app.py

    9. Check antivirus:

    10. Some antivirus software blocks keyboard hooks

    11. Add DictaPilot to exceptions

    12. Verify Python is in PATH: 

      where python

    Paste Not Working

    Problem: Text transcribes but doesn't paste

    Solutions:

    1. Try different backend: 

      PASTE_BACKEND=pynput

    2. Check clipboard: 

      # Text should be in clipboard
# Try manual paste with Ctrl+V

    3. Verify app accepts input:

    4. Some apps block programmatic input

    5. Test with Notepad first

    6. Check focus:

    7. Ensure text field is focused before releasing hotkey

    8. Click in the text field first

    9. Disable clipboard managers:

    10. Third-party clipboard managers may interfere
    11. Temporarily disable and test

    Audio Issues

    Problem: No audio input or poor quality

    Solutions:

    1. Check microphone permission:
    2. Settings → Privacy → Microphone

    3. Enable for Python/Terminal

    4. List audio devices: 

      python -c "import sounddevice; print(sounddevice.query_devices())"

    5. Select specific device: 

      # In .env
AUDIO_DEVICE=1  # Use device index from list

    6. Test microphone:

    7. Settings → System → Sound → Input
    8. Test your microphone

    9. Adjust input volume

    10. Update audio drivers:

    11. Device Manager → Sound, video and game controllers

    12. Right-click microphone → Update driver

    13. Adjust sample rate: 

      SAMPLE_RATE=44100  # Try: 16000, 44100, 48000

    Python Not Found

    Problem: python command not recognized

    Solutions:

    1. Add Python to PATH:
    2. Search "Environment Variables" in Start Menu
    3. Edit "Path" variable
    4. Add Python installation directory

    5. Example: C:\Users\YourName\AppData\Local\Programs\Python\Python312

    6. Use full path: 

      C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe app.py

    7. Reinstall Python:

    8. Download from python.org
    9. Check "Add Python to PATH" during installation

    Module Import Errors

    Problem: ModuleNotFoundError when running

    Solutions:

    1. Reinstall dependencies: 

      pip install -r requirements.txt

    2. Use virtual environment: 

      python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py

    3. Check Python version: 

      python --version
# Should be 3.10 or higher

    Antivirus Blocking

    Problem: Antivirus flags DictaPilot

    Solutions:

    1. Add to exceptions:
    2. Open your antivirus software
    3. Add DictaPilot folder to exceptions

    4. Add python.exe to exceptions

    5. Temporarily disable:

    6. Disable antivirus temporarily
    7. Test if DictaPilot works

    8. Re-enable and add exceptions

    9. Windows Defender:

    10. Settings → Update & Security → Windows Security
    11. Virus & threat protection → Manage settings
    12. Add exclusion for DictaPilot folder

    Performance Optimization

    Reduce Latency

    # Faster transcription model
GROQ_WHISPER_MODEL=whisper-large-v3-turbo

# Disable instant refine
INSTANT_REFINE=0

# Use delta paste
PASTE_MODE=delta

# Use keyboard backend (fastest)
PASTE_BACKEND=keyboard

    Reduce Resource Usage

    # Heuristic mode (no LLM)
SMART_MODE=heuristic

# Lower sample rate
SAMPLE_RATE=16000

    Auto-Start on Login

    Using Task Scheduler

    1. Open Task Scheduler:
    2. Press Win+R
    3. Type taskschd.msc

    4. Press Enter

    5. Create Basic Task:

    6. Click "Create Basic Task"
    7. Name: "DictaPilot"
    8. Trigger: "When I log on"
    9. Action: "Start a program"
    10. Program: C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe
    11. Arguments: C:\path\to\DictaPilot\app.py

    12. Start in: C:\path\to\DictaPilot

    13. Configure:

    14. Check "Run with highest privileges" if needed
    15. Finish

    Using Startup Folder

    1. Create shortcut:
    2. Right-click app.py

    3. Create shortcut

    4. Edit shortcut:

    5. Right-click shortcut → Properties
    6. Target: C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe "C:\path\to\DictaPilot\app.py"

    7. Start in: C:\path\to\DictaPilot

    8. Move to Startup:

    9. Press Win+R
    10. Type shell:startup
    11. Move shortcut to this folder

    Advanced Configuration

    Custom Hotkey Combinations

    # Modifier keys
HOTKEY=ctrl+shift+d
HOTKEY=alt+f9
HOTKEY=win+space

    Multiple Audio Devices

    # List all devices
python -c "import sounddevice; print(sounddevice.query_devices())"

# Select by index
AUDIO_DEVICE=2

# Or by name
AUDIO_DEVICE="USB Microphone"

    Network Configuration

    # Use proxy
set HTTP_PROXY=http://proxy.example.com:8080
set HTTPS_PROXY=http://proxy.example.com:8080

    Firewall Configuration

    If Windows Firewall blocks network access:

    1. Windows Defender Firewall → Allow an app
    2. Click "Change settings"
    3. Click "Allow another app"
    4. Browse to python.exe
    5. Add and enable for Private and Public networks

    Windows Version Specific

    Windows 10

    • Works with all features
    • May need admin for some apps
    • Microphone permission in Settings → Privacy

    Windows 11

    • Enhanced security features
    • Microphone permission in Settings → Privacy & security
    • May show additional security prompts
    • Recommended to use latest Python version

    Common Issues

    DLL Load Failed

    Problem: ImportError: DLL load failed

    Solution: Install Visual C++ Redistributable: - Download from Microsoft website - Install both x86 and x64 versions

    PortAudio Error

    Problem: Error loading PortAudio

    Solution: 

    pip uninstall sounddevice soundfile
pip install sounddevice soundfile

    Encoding Errors

    Problem: UnicodeDecodeError or encoding issues

    Solution: 

    # In .env
PYTHONIOENCODING=utf-8
